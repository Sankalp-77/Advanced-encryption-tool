# 🔐 Advanced Encryption Tool

## 📌 Description

The Advanced Encryption Tool is a cybersecurity application designed to securely protect sensitive files using strong encryption algorithms such as AES-256. It ensures data confidentiality by allowing users to encrypt and decrypt files, preventing unauthorized access.

---

## 🚀 Features

* 🔒 AES-256 File Encryption
* 🔓 Secure File Decryption
* 📁 Supports Multiple File Types
* 🖥️ User-Friendly GUI (Tkinter)
* 🔑 Password-Based Encryption
* ⚡ Fast and Efficient Processing
* 📝 Logging System

---

## 🛠️ Technologies Used

* Python
* Cryptography Library
* Tkinter
* OS & File Handling Modules
* Logging

---

## ⚙️ Installation

1. Clone the repository:

```id="enc002"
git clone https://github.com/your-username/advanced-encryption-tool.git
```

2. Navigate to the project folder:

```id="enc003"
cd advanced-encryption-tool
```

3. Install dependencies:

```id="enc004"
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the tool using:

```id="enc005"
python main.py
```

Select a file, enter a secure password, and choose to encrypt or decrypt.

---

## 📋 Key Functionality

* Encrypt files securely using AES-256
* Decrypt files with the correct password
* Prevent unauthorized access to sensitive data
